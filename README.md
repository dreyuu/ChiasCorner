Project Description
This system allows users to place orders online, track sales, and manage products efficiently. It is built as a capstone project to demonstrate full-stack development skills using PHP for backend, MySQL for database, and JavaScript + CSS for frontend interactivity.

Features
- User-friendly interface for placing orders.
- Product listing and search functionality.
- Order management and sales tracking.
- Responsive design compatible with desktop and mobile browsers.
- Admin panel for managing orders, products, and sales reports.

Technologies Used
- Frontend: HTML5, CSS3, JavaScript
- Backend: PHP
- Database: MySQL
- Hosting: InfinityFree (for live deployment)

Installation & Setup
1. Clone the repository:
   - git clone https://github.com/dreyuu/ChiasCorner
2. Copy the project to your local server (e.g., XAMPP htdocs folder).
3. Import the database:
   - Open phpMyAdmin.
   - Create a new database (e.g., online_ordering).
   - Import the provided .sql file in the repo.
4. Update the database connection settings in config.php (or wherever your DB credentials are stored).
5. Start your local server and open the project in a browser.

Usage
- Navigate to the homepage.
- Browse products and place orders.
- Admin can log in to manage products, orders, and sales.
